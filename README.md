# HqO BDR Marketplace

Cowork plugin marketplace for HqO GTM Engineering.

## Installation

In the Claude Desktop app (Cowork mode):

1. Click **Plugins** (bottom-left, above your name)
2. Click the **"By Anthropic"** dropdown
3. Select **"Add marketplace from GitHub"**
4. Enter: `hqo-gtm/hqo-bdr-marketplace`
5. The HqO BDR Toolkit will appear in your plugin list â€” click to install

## Plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| hqo-bdr-toolkit | 2.0.0 | Research accounts, generate outreach, create Gmail drafts |

## First Time Setup

After installing the plugin, run `/hqo:onboard` in a Cowork conversation to complete setup.
